Nombre del Proyecto:
MEDICAL APPOINTMENT APP

Objetivo General:
Diseñar e implementar un sistema de citas médicas con un backend sólido, apoyado en múltiples frameworks, con el objetivo de mejorar la eficiencia, organización y velocidad del desarrollo del sistema.

Tecnologías a Utilizar:
 - Laravel
 - Jetstream
 - Livewire
 - PHP
 - Composer
 - MySQL
 - Node.js
 - npm
 - XAMPP
